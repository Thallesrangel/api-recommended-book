# API REST de Recomendação de Livros

## Utilizadas
- Framework: Laravel
- ORM: Eloquent
- SGBD: MySQL
- Documentação da API: Swagger
- Testes: PHPUnit

Aplicando os princípios de SOLID, padrões como Services e Repository.


## Documentação
- php artisan l5-swagger:generate
- http://localhost:8000/api/docs
